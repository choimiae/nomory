# Nomory

## 📌 소개  
Nomad + Memory = Nomory(Wander Through Memories)<br>
언제 어디서든 소중한 기억을 나만의 이야기로 남길 수 있는 플랫폼

## 🛠️ 기술 스택
- **Frontend**: React, TypeScript, MUI
- **Backend**: Node.js, Express
- **Database**: MySQL

## ✨ 주요 기능
- 장소/지도 검색
- 나만의 장소를 기록/저장
- 계정 관리
